## 待生成的图标

需要以下图标文件以获得最佳体验：

### 1. 基础网站图标
- `favicon.ico` (16x16, 32x32)
- `favicon-16x16.png`
- `apple-touch-icon.png` (180x180)

### 2. PWA 图标（manifest.json 引用）
- `icon-192.png` (192x192)
- `icon-512.png` (512x512)

### 3. 社交分享图
- `og-image.png` (1200x630) - 可选，目前使用动态 API

## 快速生成方法

### 方案 A：在线工具（推荐，5分钟）

**RealFaviconGenerator** - 最完整的方案
1. 访问 https://realfavicongenerator.net/
2. 上传 `/public/favicon.svg`
3. 自定义各平台的显示效果
4. 下载生成的图标包
5. 解压后放到 `/public` 目录

**Favicon.io** - 快速简单
1. 访问 https://favicon.io/favicon-converter/
2. 上传 `/public/favicon.svg`
3. 下载图标包
4. 放到 `/public` 目录

### 方案 B：使用 Sharp 自动生成

安装依赖：
```bash
pnpm add -D sharp
```

创建生成脚本 `scripts/generate-icons.js`：
```javascript
const sharp = require('sharp')
const fs = require('fs')

const sizes = [
  { size: 16, name: 'favicon-16x16.png' },
  { size: 32, name: 'favicon-32x32.png' },
  { size: 180, name: 'apple-touch-icon.png' },
  { size: 192, name: 'icon-192.png' },
  { size: 512, name: 'icon-512.png' },
]

const svgBuffer = fs.readFileSync('./public/favicon.svg')

sizes.forEach(({ size, name }) => {
  sharp(svgBuffer)
    .resize(size, size)
    .png()
    .toFile(`./public/${name}`)
    .then(() => console.log(`✓ Generated ${name}`))
    .catch(err => console.error(`✗ Failed to generate ${name}:`, err))
})

// 生成 OG 图片
sharp(svgBuffer)
  .resize(1200, 630, { fit: 'contain', background: { r: 102, g: 126, b: 234 } })
  .png()
  .toFile('./public/og-image.png')
  .then(() => console.log('✓ Generated og-image.png'))
  .catch(err => console.error('✗ Failed to generate og-image.png:', err))
```

运行：
```bash
node scripts/generate-icons.js
```

### 方案 C：使用 Figma/Sketch 设计

如果你想定制设计：

2. 导出 SVG 和各种尺寸的 PNG
3. 放到 `/public` 目录

## 完成后

取消 `layout.tsx` 中的注释：
```typescript
icons: {
  icon: "/favicon.svg",
  shortcut: "/favicon-16x16.png",  // 取消注释
  apple: "/apple-touch-icon.png",   // 取消注释
},
```

如果生成了静态 OG 图片，可选择使用静态图：
```typescript
openGraph: {
  images: [
    {
      url: "/og-image.png",  // 改回静态图
      width: 1200,
      height: 630,
    },
  ],
}
```
